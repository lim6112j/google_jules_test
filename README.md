# Mobility on Demand Website

This project is a basic website structure for a Mobility on Demand service.

## Description

It includes:
- An `index.html` file as the main landing page.
- A `css/style.css` file for basic styling.
- A `js/script.js` file for potential future JavaScript interactivity.

## How to View

To view the website, simply open the `index.html` file in a web browser.
